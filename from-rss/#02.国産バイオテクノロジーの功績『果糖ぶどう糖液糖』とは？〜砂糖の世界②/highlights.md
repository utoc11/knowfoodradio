```
(...)

```

**キーポイント**: 分子の形を変えることで甘さが変わる驚きの技術

```

ゆと: へぇー。
(...)

TT: そう、イメチェン。
ゆと: 分子量一緒だけど、並び方が違うから、味が違うってこと？
TT: そう、味が違う。並び方っていうか、形が違うんだよね。
```
